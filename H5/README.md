## 🛠 功能模块预览
- **游戏大厅**：列表展示、桌位状态、快速加入。
- **桌内交互**：标准常规模式 vs 推推乐 (AOF) 模式切换。
- **保险机制**：完整的保险触发与操作弹窗演示。
- **表单配置**：后台接入参数的模板化选择。

## 📱 适配说明
本原型采用 **Tailwind CSS** 构建，已适配移动端 H5 视口，建议使用 Chrome 开发者工具的移动端模式或直接在手机浏览器中查看。
